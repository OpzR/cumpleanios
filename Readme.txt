Prueba realizada por Oscar Perez Rojas, Corresponde a formulario web realizado en Angular con una Api Restful Java.

Herramientas Utilizadas en el proyecto: -Spring Tool Suite Java 8, Gradle y Visual Code (desarrollo Front).

Pasos Deployar:

-Descargar Fuente repositorio Git https://github.com/OpzR/cumpleanios.git, esta descargara el back y el front de este proyecto.

-API RestFul Java: Primero debe abrir el proyecto usando Ide de Eclipse (Spring tool suite), debe tener instalado JDK 8,  importar proyecto Grade y ejecutar "run" (Ctrl+F11).	 

-Front End Angular: En la ruta del proyecto ejecutar comando npm install, abrir visual code, abrir el proyecto, por ultimo ejecturar comando ng serve.

-Abrir GoogleChrome con la siguiente url:http://localhost:4200/
